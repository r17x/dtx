# Phase 7.1: Plugin Manifest & Loader

> Dynamic plugin loading for community extensions.

---

## Goal

Enable loading plugins at runtime to extend dtx functionality.

---

## User Value

```
- Community can build extensions
- Custom backends without forking
- Shareable middleware
```

### 7.1.2 Plugin API (3h)

```rust
// crates/dtx-plugin/src/api.rs

/// Plugin entry point - called on load
pub type PluginInit = fn() -> Box<dyn Plugin>;

pub trait Plugin: Send + Sync {
    fn name(&self) -> &str;
    fn version(&self) -> &str;
    fn api_version(&self) -> u32;

    fn backends(&self) -> Vec<Box<dyn BackendPlugin>> { vec![] }
    fn middleware(&self) -> Vec<Box<dyn MiddlewarePlugin>> { vec![] }
    fn translators(&self) -> Vec<Box<dyn TranslatorPlugin>> { vec![] }
}

pub trait BackendPlugin: Send + Sync {
    fn kind(&self) -> ResourceKind;
    fn create(&self, config: serde_json::Value, event_bus: Arc<EventBus>) -> Result<Box<dyn Resource>>;
}

pub trait MiddlewarePlugin: Send + Sync {
    fn name(&self) -> &str;
    fn create(&self, config: serde_json::Value) -> Result<Box<dyn Middleware>>;
}

pub trait TranslatorPlugin: Send + Sync {
    fn from_kind(&self) -> TypeId;
    fn to_kind(&self) -> TypeId;
    fn translate(&self, from: &dyn Any) -> Result<Box<dyn Any>, TranslationError>;
    fn reverse(&self, to: &dyn Any) -> Result<Box<dyn Any>, TranslationError>;
}
```

### 7.1.3 Plugin Loader (3h)

```rust
// crates/dtx-plugin/src/loader.rs

pub struct PluginLoader {
    plugins: HashMap<String, LoadedPlugin>,
    search_paths: Vec<PathBuf>,
}

struct LoadedPlugin {
    library: libloading::Library,
    plugin: Box<dyn Plugin>,
}

impl PluginLoader {
    pub fn load(&mut self, path: &Path) -> Result<()> {
        let library = unsafe { libloading::Library::new(path)? };

        let init: libloading::Symbol<PluginInit> = unsafe {
            library.get(b"dtx_plugin_init")?
        };

        let plugin = init();

        // Verify API version
        if plugin.api_version() != CURRENT_API_VERSION {
            return Err(PluginError::IncompatibleVersion);
        }

        self.plugins.insert(plugin.name().to_string(), LoadedPlugin {
            library,
            plugin,
        });

        Ok(())
    }

    pub fn discover(&mut self) -> Result<Vec<String>> {
        let mut loaded = Vec::new();

        for search_path in &self.search_paths {
            for entry in std::fs::read_dir(search_path)? {
                let path = entry?.path();
                if path.extension().map(|e| e == "so" || e == "dylib").unwrap_or(false) {
                    if let Ok(()) = self.load(&path) {
                        loaded.push(path.display().to_string());
                    }
                }
            }
        }

        Ok(loaded)
    }

    pub fn get_backends(&self) -> Vec<Box<dyn BackendPlugin>> {
        self.plugins.values()
            .flat_map(|p| p.plugin.backends())
            .collect()
    }
}
```

### 7.1.4 Plugin Macro (2h)

```rust
// Convenience macro for plugin authors

#[macro_export]
macro_rules! dtx_plugin {
    ($plugin:ty) => {
        #[no_mangle]
        pub extern "C" fn dtx_plugin_init() -> Box<dyn dtx_plugin::Plugin> {
            Box::new(<$plugin>::new())
        }
    };
}

// Usage in plugin:
// dtx_plugin!(MyKubernetesPlugin);
```
