# Phase 6.3: Export Formats

> Export projects to docker-compose, Kubernetes, and other formats with process-compose compatibility.

---

## Goal

Implement export translators that convert dtx projects to deployment-ready formats: docker-compose.yaml, Kubernetes manifests, and process-compose.yaml. Ensure dtx is a superset of process-compose v1.x.

---

## User Value

```
- Export to docker-compose: dtx export --format docker-compose
- Export to Kubernetes: dtx export --format k8s
- Export to process-compose: dtx export --format process-compose
- Share configurations with teams using standard formats
- Deploy to production from dtx definitions
- Migration from process-compose is seamless (superset compatibility)
```

---

## Scope

```
IN SCOPE:
  - DockerComposeExporter (services, networks, volumes)
  - KubernetesExporter (Pod, Deployment, Service)
  - ProcessComposeExporter (existing format)
  - process-compose v1.x field compatibility
  - CLI export command
  - Template-based generation
  - Multi-service project export

OUT OF SCOPE:
  - Import from external formats (Phase 6.4)
  - Helm charts (future)
  - Terraform (future)
  - Cloud-specific formats (future)
```

```rust
//! Export format types and traits.

use serde::{Deserialize, Serialize};
use std::path::Path;

use super::error::{ExportError, ExportResult};

/// Supported export formats.
#[derive(Clone, Copy, Debug, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "kebab-case")]
pub enum ExportFormat {
    /// Docker Compose v3 format.
    DockerCompose,
    /// Kubernetes manifests.
    Kubernetes,
    /// Process-compose format.
    ProcessCompose,
    /// dtx native YAML.
    Dtx,
}

impl ExportFormat {
    /// Get file extension for this format.
    pub fn extension(&self) -> &'static str {
        match self {
            Self::DockerCompose => "yaml",
            Self::Kubernetes => "yaml",
            Self::ProcessCompose => "yaml",
            Self::Dtx => "yaml",
        }
    }

    /// Get default filename for this format.
    pub fn default_filename(&self) -> &'static str {
        match self {
            Self::DockerCompose => "docker-compose.yaml",
            Self::Kubernetes => "kubernetes.yaml",
            Self::ProcessCompose => "process-compose.yaml",
            Self::Dtx => "config.yaml",
        }
    }

    /// Parse from string.
    pub fn from_str(s: &str) -> Option<Self> {
        match s.to_lowercase().as_str() {
            "docker" | "docker-compose" | "compose" => Some(Self::DockerCompose),
            "k8s" | "kubernetes" | "kube" => Some(Self::Kubernetes),
            "process-compose" | "pc" => Some(Self::ProcessCompose),
            "dtx" | "native" => Some(Self::Dtx),
            _ => None,
        }
    }
}

impl std::fmt::Display for ExportFormat {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        match self {
            Self::DockerCompose => write!(f, "docker-compose"),
            Self::Kubernetes => write!(f, "kubernetes"),
            Self::ProcessCompose => write!(f, "process-compose"),
            Self::Dtx => write!(f, "dtx"),
        }
    }
}

/// Trait for exporters.
pub trait Exporter: Send + Sync {
    /// Export format.
    fn format(&self) -> ExportFormat;

    /// Export to string.
    fn export(&self, project: &ExportableProject) -> ExportResult<String>;

    /// Export to file.
    fn export_to_file(&self, project: &ExportableProject, path: &Path) -> ExportResult<()> {
        let content = self.export(project)?;
        std::fs::write(path, content).map_err(|e| ExportError::Io(e.to_string()))?;
        Ok(())
    }
}

/// Project data ready for export.
#[derive(Clone, Debug)]
pub struct ExportableProject {
    /// Project name.
    pub name: String,
    /// Project version.
    pub version: Option<String>,
    /// Services to export.
    pub services: Vec<ExportableService>,
}

/// Service data ready for export.
#[derive(Clone, Debug)]
pub struct ExportableService {
    /// Service name.
    pub name: String,
    /// Original process config.
    pub process: Option<dtx_process::config::ProcessResourceConfig>,
    /// Container config (translated).
    pub container: crate::translation::ContainerConfig,
}
```

### 6.3.2 process-compose Compatibility (2h)

**Goal:** Ensure dtx schema is a superset of process-compose v1.x

**Reference:** [process-compose v1.x documentation](https://github.com/F1bonacc1/process-compose)

#### Core Fields Mapping

| process-compose | dtx | Status | Notes |
|-----------------|-----|--------|-------|
| `command` | `command` | Identical | Shell command to run |
| `working_dir` | `working_dir` | Identical | Working directory |
| `environment` | `environment` | Identical | Environment variables (map) |
| `log_location` | `settings.log_dir` | Moved | Global or per-resource |
| `namespace` | `project.name` | Renamed | Project-level setting |
| `disabled` | `enabled: false` | Inverted | Clearer semantics |
| `is_daemon` | (inferred) | Auto | Detected from behavior |
| `shutdown.command` | `shutdown.command` | Identical | Custom shutdown command |
| `shutdown.signal` | `shutdown.signal` | Identical | Signal to send (SIGTERM) |

#### Dependency Fields Mapping

| process-compose | dtx | Status | Notes |
|-----------------|-----|--------|-------|
| `depends_on` (list) | `depends_on` (list) | Identical | Simple dependency list |
| `depends_on` (map) | `depends_on` (map) | Identical | With conditions |
| `depends_on.*.condition: process_healthy` | `healthy` | Simplified | |
| `depends_on.*.condition: process_started` | `started` | Simplified | |
| `depends_on.*.condition: process_completed_successfully` | `completed` | Simplified | |
| `availability.restart` | `restart` | Flattened | Top-level field |
| `availability.backoff_seconds` | `restart.backoff` | Extended | Supports jitter |
| `availability.max_restarts` | `restart.max_attempts` | Renamed | Clearer name |

#### Health Check Fields Mapping

| process-compose | dtx | Status | Notes |
|-----------------|-----|--------|-------|
| `readiness_probe` | `health` | Renamed | Clearer for most use cases |
| `liveness_probe` | `liveness` | Separate | dtx adds distinct liveness |
| `readiness_probe.exec` | `health.exec` | Identical | Command to run |
| `readiness_probe.http_get` | `health.http` | Simplified | Just path, host/port inferred |
| `readiness_probe.initial_delay_seconds` | `health.initial_delay` | Renamed | Duration format |
| `readiness_probe.period_seconds` | `health.interval` | Renamed | Duration format |
| `readiness_probe.timeout_seconds` | `health.timeout` | Renamed | Duration format |
| `readiness_probe.failure_threshold` | `health.retries` | Renamed | Clearer name |

#### dtx Extensions (not in process-compose)

| Field | Purpose | Example |
|-------|---------|---------|
| `kind` | Resource type | `process`, `container`, `agent` |
| `port` | Primary port | Used for health checks, display |
| `nix.packages` | Nix packages (list) | `[nodejs_20, postgresql_16]` |
| `nix.expr` | Nix expression (string) | Complex Nix environments |
| `nix.shell` | Nix shell file path | `./shell.nix` |
| `runtime` | Agent runtime | `openai`, `anthropic`, `local` |
| `model` | Agent model | `gpt-4`, `claude-3` |
| `tools` | Agent tools | List of available tools |
| `liveness` | Liveness probe | Separate from readiness |

```rust
//! Process-compose v1.x compatible exporter.

use serde::Serialize;
use std::collections::HashMap;

use super::error::{ExportError, ExportResult};
use super::types::{ExportFormat, ExportableProject, ExportableService, Exporter};

/// Process-compose exporter (v1.x compatible)
pub struct ProcessComposeExporter {
    /// Include dtx extensions as comments
    pub include_comments: bool,
}

impl Default for ProcessComposeExporter {
    fn default() -> Self {
        Self {
            include_comments: true,
        }
    }
}

impl Exporter for ProcessComposeExporter {
    fn format(&self) -> ExportFormat {
        ExportFormat::ProcessCompose
    }

    fn export(&self, project: &ExportableProject) -> ExportResult<String> {
        let pc = self.build_process_compose(project)?;
        serde_yaml::to_string(&pc)
            .map_err(|e| ExportError::Serialization(e.to_string()))
    }
}

impl ProcessComposeExporter {
    fn build_process_compose(&self, project: &ExportableProject) -> ExportResult<ProcessComposeFile> {
        let mut processes = HashMap::new();

        for svc in &project.services {
            let process = self.build_process(svc)?;
            processes.insert(svc.name.clone(), process);
        }

        Ok(ProcessComposeFile {
            version: "0.5".to_string(),
            processes,
        })
    }

    fn build_process(&self, svc: &ExportableService) -> ExportResult<ProcessComposeProcess> {
        let config = svc.process.as_ref()
            .ok_or_else(|| ExportError::Missing("process config".into()))?;

        // Map dtx depends_on to process-compose format
        let depends_on = if config.depends_on.is_empty() {
            None
        } else {
            let deps: HashMap<_, _> = config.depends_on.iter()
                .map(|d| {
                    let condition = match d.condition.as_str() {
                        "healthy" => "process_healthy",
                        "started" => "process_started",
                        "completed" => "process_completed_successfully",
                        _ => "process_started",
                    };
                    (d.target.clone(), ProcessComposeDependency {
                        condition: condition.to_string(),
                    })
                })
                .collect();
            Some(deps)
        };

        // Map dtx health to process-compose readiness_probe
        let readiness_probe = config.health.as_ref().map(|h| {
            ProcessComposeProbe {
                exec: h.exec.as_ref().map(|c| ProcessComposeExec {
                    command: c.clone(),
                }),
                http_get: h.http.as_ref().map(|path| ProcessComposeHttpGet {
                    path: path.clone(),
                    port: config.port.unwrap_or(8080),
                }),
                initial_delay_seconds: parse_duration(&h.initial_delay),
                period_seconds: Some(parse_duration_or(&h.interval, 10)),
                timeout_seconds: Some(parse_duration_or(&h.timeout, 5)),
                failure_threshold: Some(h.retries),
            }
        });

        // Map dtx restart to process-compose availability
        let availability = config.restart.as_ref().map(|r| ProcessComposeAvailability {
            restart: r.policy.clone(),
            backoff_seconds: parse_duration(&r.backoff),
            max_restarts: r.max_attempts,
        });

        Ok(ProcessComposeProcess {
            command: config.command.clone(),
            working_dir: config.working_dir.clone(),
            environment: if config.environment.is_empty() {
                None
            } else {
                Some(config.environment.clone())
            },
            depends_on,
            readiness_probe,
            availability,
            disabled: Some(!config.enabled),
        })
    }
}

// Helper to parse duration strings like "5s" → 5
fn parse_duration(s: &Option<String>) -> Option<u32> {
    s.as_ref().and_then(|d| {
        d.trim_end_matches('s').parse().ok()
    })
}

fn parse_duration_or(s: &str, default: u32) -> u32 {
    s.trim_end_matches('s').parse().unwrap_or(default)
}
```

```rust
//! Docker Compose exporter.

use serde::Serialize;
use std::collections::HashMap;

use super::error::{ExportError, ExportResult};
use super::types::{ExportFormat, ExportableProject, ExportableService, Exporter};
use crate::translation::{
    ContainerConfig, ContainerHealthCheck, ContainerRestartPolicy,
    DependencyCondition, HealthCheckTest, PortMapping, VolumeMount,
};

/// Docker Compose v3 exporter.
pub struct DockerComposeExporter {
    /// Compose file version.
    pub version: String,
    /// Include build context.
    pub include_build: bool,
}

impl Default for DockerComposeExporter {
    fn default() -> Self {
        Self {
            version: "3.8".to_string(),
            include_build: false,
        }
    }
}

impl Exporter for DockerComposeExporter {
    fn format(&self) -> ExportFormat {
        ExportFormat::DockerCompose
    }

    fn export(&self, project: &ExportableProject) -> ExportResult<String> {
        let compose = self.build_compose(project)?;

        serde_yaml::to_string(&compose)
            .map_err(|e| ExportError::Serialization(e.to_string()))
    }
}

impl DockerComposeExporter {
    /// Create new exporter.
    pub fn new() -> Self {
        Self::default()
    }

    /// Set compose version.
    pub fn with_version(mut self, version: impl Into<String>) -> Self {
        self.version = version.into();
        self
    }

    fn build_compose(&self, project: &ExportableProject) -> ExportResult<ComposeFile> {
        let mut services = HashMap::new();

        for svc in &project.services {
            let compose_service = self.build_service(&svc.container)?;
            services.insert(svc.name.clone(), compose_service);
        }

        Ok(ComposeFile {
            version: Some(self.version.clone()),
            name: Some(project.name.clone()),
            services,
            networks: None,
            volumes: None,
        })
    }

    fn build_service(&self, container: &ContainerConfig) -> ExportResult<ComposeService> {
        // Build depends_on
        let depends_on = if container.depends_on.is_empty() {
            None
        } else {
            let deps: HashMap<_, _> = container
                .depends_on
                .iter()
                .map(|d| {
                    let condition = match d.condition {
                        DependencyCondition::ServiceStarted => "service_started",
                        DependencyCondition::ServiceHealthy => "service_healthy",
                        DependencyCondition::ServiceCompletedSuccessfully => "service_completed_successfully",
                    };
                    (d.service.clone(), ComposeDependsOn { condition: condition.to_string() })
                })
                .collect();
            Some(deps)
        };

        // Build ports
        let ports = if container.ports.is_empty() {
            None
        } else {
            Some(
                container
                    .ports
                    .iter()
                    .map(format_port)
                    .collect(),
            )
        };

        // Build volumes
        let volumes = if container.volumes.is_empty() {
            None
        } else {
            Some(
                container
                    .volumes
                    .iter()
                    .map(format_volume)
                    .collect(),
            )
        };

        // Build environment
        let environment = if container.environment.is_empty() {
            None
        } else {
            Some(container.environment.clone())
        };

        // Build healthcheck
        let healthcheck = container.health_check.as_ref().map(|hc| {
            ComposeHealthCheck {
                test: format_health_test(&hc.test),
                interval: Some(hc.interval.clone()),
                timeout: Some(hc.timeout.clone()),
                retries: Some(hc.retries),
                start_period: hc.start_period.clone(),
            }
        });

        // Build labels
        let labels = if container.labels.is_empty() {
            None
        } else {
            Some(container.labels.clone())
        };

        // Build deploy (for resource limits)
        let deploy = container.resources.as_ref().map(|r| {
            ComposeDeploy {
                resources: Some(ComposeResources {
                    limits: Some(ComposeResourceLimit {
                        cpus: r.cpus.clone(),
                        memory: r.memory.clone(),
                    }),
                    reservations: None,
                }),
            }
        });

        Ok(ComposeService {
            image: Some(container.image.clone()),
            container_name: Some(container.id.as_str().to_string()),
            command: container.command.clone(),
            entrypoint: container.entrypoint.clone(),
            working_dir: container.working_dir.clone(),
            environment,
            ports,
            volumes,
            depends_on,
            restart: Some(format_restart(&container.restart)),
            healthcheck,
            networks: container.network.as_ref().map(|n| vec![n.clone()]),
            labels,
            deploy,
            build: None,
        })
    }
}

// Helper functions

fn format_port(port: &PortMapping) -> String {
    if port.host == port.container {
        format!("{}", port.host)
    } else {
        format!("{}:{}", port.host, port.container)
    }
}

fn format_volume(vol: &VolumeMount) -> String {
    let source = vol.source.to_string_lossy();
    if vol.read_only {
        format!("{}:{}:ro", source, vol.target)
    } else {
        format!("{}:{}", source, vol.target)
    }
}

fn format_restart(policy: &ContainerRestartPolicy) -> String {
    match policy {
        ContainerRestartPolicy::No => "no".to_string(),
        ContainerRestartPolicy::Always => "always".to_string(),
        ContainerRestartPolicy::OnFailure => "on-failure".to_string(),
        ContainerRestartPolicy::UnlessStopped => "unless-stopped".to_string(),
    }
}
```

```rust
//! Kubernetes manifest exporter.

use serde::Serialize;
use std::collections::HashMap;

use super::error::{ExportError, ExportResult};
use super::types::{ExportFormat, ExportableProject, ExportableService, Exporter};
use crate::translation::{ContainerConfig, ContainerHealthCheck, HealthCheckTest};

/// Kubernetes manifest exporter.
pub struct KubernetesExporter {
    /// API version for Deployment.
    pub api_version: String,
    /// Default namespace.
    pub namespace: Option<String>,
    /// Generate Service resources.
    pub generate_services: bool,
    /// Use Deployment (vs Pod).
    pub use_deployment: bool,
}

impl Default for KubernetesExporter {
    fn default() -> Self {
        Self {
            api_version: "apps/v1".to_string(),
            namespace: None,
            generate_services: true,
            use_deployment: true,
        }
    }
}

impl Exporter for KubernetesExporter {
    fn format(&self) -> ExportFormat {
        ExportFormat::Kubernetes
    }

    fn export(&self, project: &ExportableProject) -> ExportResult<String> {
        let mut documents = Vec::new();

        for svc in &project.services {
            if self.use_deployment {
                let deployment = self.build_deployment(project, svc)?;
                let yaml = serde_yaml::to_string(&deployment)
                    .map_err(|e| ExportError::Serialization(e.to_string()))?;
                documents.push(yaml);
            } else {
                let pod = self.build_pod(svc)?;
                let yaml = serde_yaml::to_string(&pod)
                    .map_err(|e| ExportError::Serialization(e.to_string()))?;
                documents.push(yaml);
            }

            if self.generate_services && !svc.container.ports.is_empty() {
                let k8s_service = self.build_service(project, svc)?;
                let yaml = serde_yaml::to_string(&k8s_service)
                    .map_err(|e| ExportError::Serialization(e.to_string()))?;
                documents.push(yaml);
            }
        }

        Ok(documents.join("---\n"))
    }
}

impl KubernetesExporter {
    /// Create new exporter.
    pub fn new() -> Self {
        Self::default()
    }

    /// Set namespace.
    pub fn with_namespace(mut self, ns: impl Into<String>) -> Self {
        self.namespace = Some(ns.into());
        self
    }

    fn build_deployment(
        &self,
        project: &ExportableProject,
        svc: &ExportableService,
    ) -> ExportResult<K8sDeployment> {
        let container = self.build_container(&svc.container)?;

        let labels = common_labels(&project.name, &svc.name);

        Ok(K8sDeployment {
            api_version: self.api_version.clone(),
            kind: "Deployment".to_string(),
            metadata: K8sMetadata {
                name: svc.name.clone(),
                namespace: self.namespace.clone(),
                labels: Some(labels.clone()),
            },
            spec: K8sDeploymentSpec {
                replicas: 1,
                selector: K8sSelector {
                    match_labels: labels.clone(),
                },
                template: K8sPodTemplateSpec {
                    metadata: K8sMetadata {
                        name: svc.name.clone(),
                        namespace: None,
                        labels: Some(labels),
                    },
                    spec: K8sPodSpec {
                        containers: vec![container],
                        restart_policy: None, // Deployment handles this
                    },
                },
            },
        })
    }

    fn build_pod(&self, svc: &ExportableService) -> ExportResult<K8sPod> {
        let container = self.build_container(&svc.container)?;

        let restart_policy = match svc.container.restart {
            crate::translation::ContainerRestartPolicy::Always => Some("Always".to_string()),
            crate::translation::ContainerRestartPolicy::OnFailure => Some("OnFailure".to_string()),
            crate::translation::ContainerRestartPolicy::No => Some("Never".to_string()),
            _ => Some("Always".to_string()),
        };

        Ok(K8sPod {
            api_version: "v1".to_string(),
            kind: "Pod".to_string(),
            metadata: K8sMetadata {
                name: svc.name.clone(),
                namespace: self.namespace.clone(),
                labels: None,
            },
            spec: K8sPodSpec {
                containers: vec![container],
                restart_policy,
            },
        })
    }

    fn build_container(&self, config: &ContainerConfig) -> ExportResult<K8sContainer> {
        // Build env vars
        let env = if config.environment.is_empty() {
            None
        } else {
            Some(
                config
                    .environment
                    .iter()
                    .map(|(k, v)| K8sEnvVar {
                        name: k.clone(),
                        value: Some(v.clone()),
                        value_from: None,
                    })
                    .collect(),
            )
        };

        // Build ports
        let ports = if config.ports.is_empty() {
            None
        } else {
            Some(
                config
                    .ports
                    .iter()
                    .map(|p| K8sContainerPort {
                        container_port: p.container as i32,
                        protocol: match p.protocol {
                            crate::translation::Protocol::Tcp => "TCP".to_string(),
                            crate::translation::Protocol::Udp => "UDP".to_string(),
                        },
                    })
                    .collect(),
            )
        };

        // Build probes
        let liveness_probe = config.health_check.as_ref().map(|hc| build_probe(hc));
        let readiness_probe = liveness_probe.clone();

        // Build resources
        let resources = config.resources.as_ref().map(|r| K8sResources {
            limits: Some(K8sResourceQuantity {
                cpu: r.cpus.clone(),
                memory: r.memory.clone(),
            }),
            requests: None,
        });

        Ok(K8sContainer {
            name: config.id.as_str().to_string(),
            image: config.image.clone(),
            command: config.entrypoint.clone(),
            args: config.command.clone(),
            working_dir: config.working_dir.clone(),
            env,
            ports,
            liveness_probe,
            readiness_probe,
            resources,
        })
    }

    fn build_service(
        &self,
        project: &ExportableProject,
        svc: &ExportableService,
    ) -> ExportResult<K8sService> {
        let ports: Vec<_> = svc
            .container
            .ports
            .iter()
            .map(|p| K8sServicePort {
                port: p.host as i32,
                target_port: p.container as i32,
                protocol: "TCP".to_string(),
            })
            .collect();

        let labels = common_labels(&project.name, &svc.name);

        Ok(K8sService {
            api_version: "v1".to_string(),
            kind: "Service".to_string(),
            metadata: K8sMetadata {
                name: svc.name.clone(),
                namespace: self.namespace.clone(),
                labels: Some(labels.clone()),
            },
            spec: K8sServiceSpec {
                selector: labels,
                ports,
                service_type: None, // ClusterIP by default
            },
        })
    }
}

fn common_labels(project: &str, service: &str) -> HashMap<String, String> {
    let mut labels = HashMap::new();
    labels.insert("app.kubernetes.io/name".to_string(), service.to_string());
    labels.insert("app.kubernetes.io/part-of".to_string(), project.to_string());
    labels.insert("app.kubernetes.io/managed-by".to_string(), "dtx".to_string());
    labels
}

fn build_probe(health: &ContainerHealthCheck) -> K8sProbe {
    let exec = match &health.test {
        HealthCheckTest::Cmd(args) => Some(K8sExecAction { command: args.clone() }),
        HealthCheckTest::CmdShell(cmd) => Some(K8sExecAction {
            command: vec!["sh".to_string(), "-c".to_string(), cmd.clone()],
        }),
    };

    // Parse interval/timeout
    fn parse_seconds(s: &str) -> i32 {
        s.trim_end_matches('s').parse().unwrap_or(10)
    }

    K8sProbe {
        exec,
        http_get: None,
        tcp_socket: None,
        initial_delay_seconds: health
            .start_period
            .as_ref()
            .map(|s| parse_seconds(s)),
        period_seconds: Some(parse_seconds(&health.interval)),
        timeout_seconds: Some(parse_seconds(&health.timeout)),
        failure_threshold: Some(health.retries as i32),
        success_threshold: Some(1),
    }
}
```

```rust
//! Export command implementation.

use anyhow::Result;
use clap::Args;
use std::path::PathBuf;

use dtx_core::export::{
    DockerComposeExporter, ExportFormat, ExportableProject, ExportableService,
    Exporter, KubernetesExporter,
};
use dtx_core::translation::{default_registry, TranslationContext};
use dtx_process::config::ProcessResourceConfig;

use crate::context::CmdContext;

#[derive(Args)]
pub struct ExportArgs {
    /// Export format.
    #[arg(short, long, default_value = "docker-compose")]
    pub format: String,

    /// Output file (default: stdout or format-specific name).
    #[arg(short, long)]
    pub output: Option<PathBuf>,

    /// Specific services to export (default: all).
    #[arg(short, long)]
    pub services: Option<Vec<String>>,

    /// Kubernetes namespace.
    #[arg(long)]
    pub namespace: Option<String>,

    /// Use Pods instead of Deployments (k8s only).
    #[arg(long)]
    pub pods: bool,

    /// Default image for services without inferred image.
    #[arg(long)]
    pub default_image: Option<String>,
}

impl ExportArgs {
    pub async fn run(&self, ctx: &CmdContext) -> Result<()> {
        let format = ExportFormat::from_str(&self.format)
            .ok_or_else(|| anyhow::anyhow!("Unknown format: {}", self.format))?;

        // Load project services from database
        let project = ctx.load_current_project().await?;
        let services = ctx.service_repo.find_by_project(&project.id).await?;

        // Filter services if specified
        let services: Vec<_> = if let Some(filter) = &self.services {
            services
                .into_iter()
                .filter(|s| filter.contains(&s.name))
                .collect()
        } else {
            services
        };

        // Build translation context
        let mut translation_ctx = TranslationContext::new();
        if let Some(image) = &self.default_image {
            translation_ctx = translation_ctx.default_value("image", image.clone());
        }

        // Translate services
        let registry = default_registry();
        let mut exportable_services = Vec::new();

        for service in services {
            let process_config = service.to_process_config()?;
            let container = registry.translate_with_context(&process_config, &translation_ctx)?;

            exportable_services.push(ExportableService {
                name: service.name.clone(),
                process: Some(process_config),
                container,
            });
        }

        let exportable = ExportableProject {
            name: project.name.clone(),
            version: None,
            services: exportable_services,
        };

        // Create exporter and export
        let output = match format {
            ExportFormat::DockerCompose => {
                let exporter = DockerComposeExporter::new();
                exporter.export(&exportable)?
            }
            ExportFormat::Kubernetes => {
                let mut exporter = KubernetesExporter::new();
                if let Some(ns) = &self.namespace {
                    exporter = exporter.with_namespace(ns);
                }
                if self.pods {
                    exporter.use_deployment = false;
                }
                exporter.export(&exportable)?
            }
            ExportFormat::ProcessCompose => {
                // Use existing process-compose export
                ctx.yaml_generator.generate(&services)?
            }
            ExportFormat::Dtx => {
                // Native format
                serde_yaml::to_string(&exportable)?
            }
        };

        // Write output
        if let Some(path) = &self.output {
            std::fs::write(path, &output)?;
            println!("Exported to {}", path.display());
        } else {
            println!("{}", output);
        }

        Ok(())
    }
}
```
